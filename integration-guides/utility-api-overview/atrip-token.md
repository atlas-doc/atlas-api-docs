# ATRIP Token

Use this page when your workflow needs an ATRIP token.

This page uses `ATRIP Token` as the primary name.

You may also see this API called `getAtripToken`.

### Main API

* `getAtripToken.do`

### Use this when you need

* Launch ATRIP from Atlas-integrated workflows
* Generate temporary access for ATRIP features
* Support secure downstream handoff
